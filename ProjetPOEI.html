This simplified diagram summarizes my end of POEI DevOps training project.
On behalf of a small fictitious company, it was necessary: ​​
  • Set up Linux servers (Ubuntu, CentOS, etc.) to host company services. (Blogs, Gitlab, etc.)
  • Implement monitoring solutions to monitor the status of virtual machines, containers, networks, servers, etc., and receive alerts in the event of a problem. (Prometheus Grafana Stack)
  • Priorize the use of Docker for the deployment of services.
  • Priorize the use of Ansible for server configuration.
Another part of the project, which you will find at the bottom right, focuses more on Windows administration and the following tasks:
  • Networking
  • Scripting
  • Management of GPOs, users and AD
